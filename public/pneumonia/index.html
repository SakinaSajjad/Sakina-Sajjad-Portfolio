Pneumonia Disease Detection System

AI-powered cross-platform mobile healthcare app using chest X-ray image analysis

In Active Development

Overview

Building a cross-platform mobile application using React Native with a FastAPI (Python) backend. The system integrates an EfficientNetB3 deep learning model for pneumonia detection from chest X-ray images, with Firebase, OpenAI API, and Google Maps API for healthcare features.

React Native FastAPI EfficientNetB3 Firebase OpenAI API Google Maps

Key Features (Planned)

  • Upload and analyze chest X-ray images for pneumonia detection
  • AI-powered diagnosis using EfficientNetB3 model
  • Healthcare assistant powered by OpenAI API
  • Location-based hospital finder via Google Maps
  • Secure authentication and data storage with Firebase
