Pneumonia Detection FYP

Academic research project — React Native app with ML-assisted image analysis (not a medical service)

In Active Development

This is a university Final Year Project by a CS student for learning purposes only. It is not a licensed medical device and does not provide clinical diagnosis.

Overview

Building a cross-platform mobile application using React Native with a FastAPI (Python) backend. The project explores EfficientNetB3 for chest X-ray image classification as part of academic coursework, with Firebase, OpenAI API, and Google Maps integration.

React Native FastAPI EfficientNetB3 Firebase OpenAI API

Planned Features

  • Image upload and ML model inference for learning demo
  • FastAPI backend for model serving
  • Firebase authentication and storage
  • Hospital locator via Google Maps API
